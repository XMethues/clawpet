# clawchat-pet

clawchat-pet 把 Hermes 中的 Agent 活动解释为宠物持续发展的修炼体验，并把当前宠物的活动呈现给主人。宠物主要来自 Petdex，`yinyue-2` 是默认宠物而不是特殊领域角色。

## Language

**Hermes 活动**:
Agent 在 Hermes 中发生的可观察行为或生命周期变化，是修炼事件和活动状态的共同来源。
_Avoid_: 修炼事件、活动状态

**宠物**:
承载共享修炼进度和活动呈现的当前角色；宠物主要来自 Petdex，未选择时使用 `yinyue-2`，切换宠物不会重置进度。
_Avoid_: 银月

**Petdex**:
clawchat-pet 的主要宠物来源。

**修炼进度**:
独立于当前宠物的持久修炼旅程，由所有宠物共享。
_Avoid_: 宠物存档

**宠物话语**:
当前宠物以文字气泡呈现的话语，使用 Petdex 的 `displayName` 作为说话者；它不是音频。
_Avoid_: 宠物语音

**宠物人格**:
按 Petdex 宠物分别保存、由主人修改或生成的话语风格；它只影响宠物话语，不改变修炼规则或进度。切换到尚未决定人格的宠物时，应询问主人是否创建；若主人跳过，该宠物使用中立话语且不重复询问。

**修炼事件**:
由一次 Hermes 活动解释而来的一次性领域事实；它只结算一次，可能改变修炼进度、状态或纪事。
_Avoid_: 活动状态

**活动状态**:
当前宠物所有在途活动的综合短时呈现，而非最后一条事件的标签；它可以随时间自然老化，但老化本身不会产生修炼事件。
_Avoid_: 修炼事件

**候旨**:
等待主人作出人工批准决定时的活动状态；smart-mode 自动判断不属于候旨。
_Avoid_: 自动批准判断

**推演**:
Agent 正在思考或进行 smart-mode 自动判断时的活动状态。
_Avoid_: 候旨

**分神化身**:
子 Agent 正在代为执行任务时的活动状态；它本身不产生额外修炼收益，收益由父级委派活动结算。
_Avoid_: 独立修炼事件

**历练失败**:
工具实际执行后发生错误所形成的修炼事件；主人拒绝批准或批准超时不属于历练失败。
_Avoid_: 批准拒绝、批准超时

**历练结果未明**:
工具已经结束，但现有结果不足以可靠判断成功或失败；它不产生奖励或惩罚。
_Avoid_: 历练成功、历练失败
