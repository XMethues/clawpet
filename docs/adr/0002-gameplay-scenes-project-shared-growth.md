# 玩法场景只投影共享成长

clawchat-pet 只维护一份由所有宠物和玩法场景共享的成长进度；它的持久结构与规则标识均使用场景中立词汇。稳定成长事实只包含场景中立的语义标识、数值和关系，不包含任何玩法场景的名称、话语、提示或纪事文本。共享成长 module 提供当前成长快照与有界的近期稳定成长事实，不保存并重放全部事件来重建进度。玩法场景只消费这些只读稳定结果，不接触持久结构或结算内部状态；它在成长事件完成去重和结算后，才把稳定成长事实投影为主题名称、话语和纪事。每个玩法场景拥有多个视觉皮肤和一个默认皮肤，皮肤只接受视觉覆盖。切换场景会选择该场景的默认皮肤，但不会重置、复制、重放或改变已经结算的成长。若未来需要不同公式，它应成为独立规则模块而不是玩法场景或皮肤能力。
